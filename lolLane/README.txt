③ 플레이어 프리셋 불러올 때 프리셋 코드를 입력하지 않으면 기본 프리셋 (Riot ID 제거 버전) 로드되도록 변경됩니다.
④ [4~5인] 순차 선택 모드에서 픽 순서 결정 로직이 개선됩니다.
  - 동일 플레이어가 꼴등에 배정될 확률이 보정치에 의해 감소됩니다.
  - 보정치는 해당 플레이어가 꼴등에 연속으로 배정될수록 증가됩니다.

  - 해당 변경 사항은 다른 순차 선택 모드에서는 적용되지 않습니다.
⑤ 기타 UI가 개선됩니다.
